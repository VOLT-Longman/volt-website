# VOLT 웹사이트 수동 QA 체크리스트

배포 전 최신 브라우저에서 다크/라이트 테마를 각각 한 번씩 확인합니다.

## 기본 진입 / 내비게이션

- [ ] 홈 화면이 정상 로드된다.
- [ ] F5 새로고침 시 기본 홈 화면으로 복귀한다.
- [ ] 헤더 메뉴에서 `무역플래너`로 이동할 수 있다.
- [ ] 모바일 메뉴에서 `무역플래너`로 이동할 수 있다.
- [ ] 모바일 메뉴를 열면 핵심 메뉴 `소개 / 함선DB / 무역플래너 / 무역허브 / 공지 / 가입하기`가 먼저 보인다.
- [ ] 모바일 메뉴 상단 `← 닫기`, `×`, 하단 `메뉴 닫기`가 모두 동작한다.
- [ ] 데스크톱 헤더의 우측 강조 버튼 `가입하기`가 `#join`으로 이동한다.
- [ ] 무역가이드의 `무역플래너 열기` CTA가 정상 동작한다.
- [ ] 데스크톱 `더보기` 메뉴와 모바일 메뉴가 정상적으로 열리고 닫힌다.

## 함선DB

- [ ] 전체 검색에서 함선을 검색하면 해당 함선 모달이 열린다.
- [ ] 함선 검색어, 태그 필터, 제조사 필터, `미구현 제외`, 정렬이 각각 동작한다.
- [ ] 목적별 추천 적용 시 요약 카드가 나타난다.
- [ ] 목적별 추천 해제 시 요약 카드가 사라진다.
- [ ] 함선을 비교 목록에 추가/제거할 수 있다.
- [ ] 2척 이상 비교 시 비교 모달과 비교 요약이 정상 표시된다.

## 무역플래너 — 함선 picker

- [ ] 함선 picker에서 함선명으로 검색된다.
- [ ] 함선 picker 결과 목록이 뒤쪽 글자와 겹치지 않고 독립된 패널처럼 보인다.
- [ ] 제조사, 역할, 크기, cargo 값, 태그로도 검색된다.
- [ ] 결과가 없을 때 자연스러운 안내 문구가 표시된다.
- [ ] 함선 선택 후 요약 카드에 함선명, 제조사, cargo, size, 추천 운용 문구가 표시된다.
- [ ] 함선 선택 시 숨김 select 값과 계산 결과가 함께 갱신된다.
- [ ] 입력값이 비어 있거나 0일 때 함선 cargo 값이 기본 운송량으로 반영된다.
- [ ] 미구현 함선이 무역플래너 후보에 나오지 않는다.
- [ ] cargo 0 함선이 무역플래너 후보에 나오지 않는다.
- [ ] 포커스 시 결과가 열리고 `ArrowDown`, `ArrowUp`, `Enter`, `Escape`가 동작한다.
- [ ] 바깥 클릭 시 결과 목록이 닫힌다.

## 무역플래너 — 계산 / 프리셋

- [ ] 모든 작전 프리셋이 적용되고 적절한 함선이 자동 선택된다.
- [ ] 적합도, 판단 사유, 역할 배분, 선택 함선 적합도 카드가 표시된다.
- [ ] 고위험 + 호위 부족 조건에서 `비추천` 또는 경고가 표시된다.
- [ ] 3회 이상 출격이면 추가 화물선 권장 문구가 표시된다.
- [ ] 브리핑 복사 시 최신 계산 결과가 포함되고 토스트가 표시된다.

## UEX 상품 picker

- [ ] UEX 상품 목록 로딩 중 상태가 표시된다.
- [ ] UEX 상품 검색 영역이 플래너 입력 흐름 가까이에 표시된다.
- [ ] 상품명, code, category_name으로 검색된다.
- [ ] 결과가 없을 때 영문 상품명/코드 재검색 안내가 표시된다.
- [ ] 상품 선택 전 후보 조회 버튼은 비활성화되어 있다.
- [ ] 상품 선택 후 요약 카드에 상품명, code/category, 준비 완료 문구가 표시된다.
- [ ] 후보 조회가 정상 동작한다.
- [ ] UEX API 실패 시 fallback 문구가 표시된다.
- [ ] UEX 상품 선택 후 거래 후보 정보가 무역플래너 결과와 Discord 브리핑에 반영된다.

## 함선DB ↔ 무역플래너 연결

- [ ] 함선 카드의 `무역 플래너에서 사용` 버튼이 `#trade-planner`로 이동시킨다.
- [ ] 함선 카드에서 이동 시 해당 함선이 선택되고 cargo 값이 자동 반영된다.
- [ ] 함선 상세 모달에서 이동 시 모달이 닫히고 플래너가 자동 계산된다.
- [ ] 함선 비교 모달에서 특정 함선을 플래너로 넘길 수 있다.
- [ ] 플래너 결과의 `함선 상세 보기` 버튼이 현재 입력값을 유지한 채 모달을 연다.
- [ ] 미구현 또는 `plannerEligible: false` 함선에는 플래너 전달 버튼이 보이지 않는다.

## 콘텐츠 섹션

- [ ] 갤러리 데이터가 비어 있으면 카드형 빈 상태가 자연스럽게 표시된다.
- [ ] 공지 카드를 클릭하면 상세 모달이 열린다.
- [ ] 공지 링크 복사 버튼이 동작한다.
- [ ] FAQ 항목이 열리고 닫힌다.
- [ ] 정책 링크 복사가 동작한다.

## 테마 / 접근성 / 반응형

- [ ] 라이트/다크 테마 전환이 정상 동작한다.
- [ ] picker 입력창, 결과 목록, 선택 요약 카드의 대비가 두 테마에서 충분하다.
- [ ] 키보드 이동 시 `focus-visible` 표시가 유지된다.
- [ ] 모바일 폭에서 picker 결과 목록이 화면 밖으로 넘치지 않는다.
- [ ] 모바일 폭에서 추천 함선 영역은 기본 접힘 상태이며 필요할 때만 펼쳐진다.
- [ ] 모바일 폭에서 무역플래너 카드와 UEX 카드가 깨지지 않는다.
