# VOLT 웹사이트 수동 QA 체크리스트

배포 전 최신 브라우저에서 다크/라이트 테마를 각각 한 번씩 확인한다.

## 기본 진입

- [ ] 홈 화면이 정상 로드된다.
- [ ] F5 새로고침 시 기본 홈 화면으로 복귀한다.
- [ ] 상단 메뉴 이동이 정상 동작한다.
- [ ] 데스크톱 `더보기` 메뉴가 열리고 닫힌다.
- [ ] 모바일 메뉴가 열리고 닫히며 키보드 포커스가 메뉴 안에 머문다.

## 검색 / 함선 DB

- [ ] 전체 검색에서 함선을 검색하면 해당 함선 모달이 열린다.
- [ ] 함선 검색어, 태그 필터, 제조사 필터, `미구현 제외`, 정렬이 각각 동작한다.
- [ ] 목적별 추천을 적용하면 요약 카드가 나타난다.
- [ ] 목적별 추천을 해제하면 요약 카드가 사라진다.
- [ ] 함선을 비교 목록에 추가하고 제거할 수 있다.
- [ ] 2척 이상 선택 후 비교 모달이 열리고 비교 요약이 표시된다.

## 무역 플래너

- [ ] 각 작전 유형에서 계산 결과가 생성된다.
- [ ] 적합도, 판단 사유, 역할 배분, 함선 적합도 카드가 표시된다.
- [ ] 고위험 + 호위 부족 조건에서 `비추천`이 표시된다.
- [ ] 브리핑 복사 버튼이 최신 계산 결과를 복사하고 토스트를 표시한다.

## 콘텐츠 섹션

- [ ] 갤러리 데이터가 비어 있을 때 빈 상태 카드가 자연스럽게 표시된다.
- [ ] 공지 카드를 클릭하면 상세 모달이 열린다.
- [ ] 공지 링크 복사 버튼이 동작한다.
- [ ] FAQ 항목이 열리고 닫힌다.
- [ ] 정책 링크 복사가 동작한다.

## 테마 / 접근성

- [ ] 라이트/다크 테마 전환이 정상 동작한다.
- [ ] 키보드 탭 이동 시 `focus-visible` 표시가 사라지지 않는다.
- [ ] 모바일 폭에서도 함선 추천, 비교 모달, 무역 플래너가 깨지지 않는다.
